系统帮助目录说明：

(2)目录：PROG

     该目录下，放置每个程序的操作说明文档；

     文档名：程序代码.htm

     触发条件：当用户打开程序，在程序右上角点击帮助按钮是触发,

               系统弹出窗口,显示；该文档内容;


(1)目录：MENU

     该目录下，放置每个菜单项的导航文档；

     文档名：菜单项.htm

     触发条件：在用户属性中设置了参数:showmenudirhelp=yes;
               
               当点击菜单项时，触发,系统在导航页面上显示该文档内容；

(3)程序页面：PGPG

     该目录下，放置程序相关页面；

     文档名：[程序指定的文件].htm

     触发条件：当用户打开程序,程序调用显示
